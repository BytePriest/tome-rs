# tome-rs

基于 Rust 和 Ratatui 的终端文件浏览器与文本编辑器。

## 快捷键

### 全局
| 按键 | 功能 |
|------|------|
| `Ctrl+Q` / `Ctrl+C` | 退出程序 |
| `Tab` | 切换焦点（文件树 ↔ 编辑区） |

### 文件树（左侧面板）
| 按键 | 功能 |
|------|------|
| `↑` / `↓` | 上下移动选择 |
| `Enter` | 打开文件（加载到编辑器）/ 展开或收起目录 |

### 编辑器（右侧面板）
| 按键 | 功能 |
|------|------|
| `↑` / `↓` / `←` / `→` | 移动光标 |
| `Enter` | 换行 |
| `Backspace` | 删除字符 |
| 输入任意字符 | 插入文本 |
| `Ctrl+S` | 保存当前文件 |

### 搜索（`Ctrl+F`）
| 按键 | 功能 |
|------|------|
| `Ctrl+F` | 打开搜索栏（编辑器底部） |
| 输入文字 | 键入搜索关键词 |
| `Enter` | 执行搜索，跳转到光标后的第一个匹配 |
| `Alt+N` | 下一个匹配 |
| `Alt+P` | 上一个匹配 / 从光标之前向上搜索 |
| `Esc` | 关闭搜索栏 |
